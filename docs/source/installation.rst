.. _installation:

Installation
============

Basic Installation
------------------

.. code-block:: bash

   pip install prismalog

With YAML Support
-----------------

.. code-block:: bash

   pip install prismalog[yaml]

With All Features
-----------------

.. code-block:: bash

   pip install prismalog[all]

For Development
---------------

.. code-block:: bash

   pip install prismalog[dev]
